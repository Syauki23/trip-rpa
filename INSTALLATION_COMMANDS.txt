=================================================================
TRIP SYSTEM - COMPLETE INSTALLATION COMMANDS
=================================================================

STEP 1: ENVIRONMENT SETUP
-----------------------------------------------------------------
copy .env.example .env

Then edit .env file and configure:
DB_DATABASE=trip_system
DB_USERNAME=root
DB_PASSWORD=

STEP 2: GENERATE APPLICATION KEY
-----------------------------------------------------------------
php artisan key:generate

STEP 3: RUN MIGRATIONS
-----------------------------------------------------------------
php artisan migrate

Expected output:
- 2014_10_12_000000_create_users_table ................... DONE
- 2014_10_12_100000_create_password_reset_tokens_table ... DONE
- 2019_08_19_000000_create_failed_jobs_table ............. DONE
- 2019_12_14_000001_create_personal_access_tokens_table .. DONE
- 2024_01_01_000000_create_roles_table ................... DONE
- 2024_01_01_000001_add_role_id_to_users_table ........... DONE
- 2024_01_01_000002_create_vehicles_table ................ DONE
- 2026_02_23_075611_create_trips_table ................... DONE

STEP 4: SEED DATABASE
-----------------------------------------------------------------
php artisan db:seed

Expected output:
- RoleSeeder ............................................. DONE
- UserSeeder ............................................. DONE

This creates:
✓ 3 roles: admin, supervisor, driver
✓ 3 users: admin@pt.com, supervisor@pt.com, driver@pt.com

STEP 5: CREATE STORAGE LINK
-----------------------------------------------------------------
php artisan storage:link

Expected output:
The [public/storage] link has been connected to [storage/app/public].

STEP 6: START DEVELOPMENT SERVER
-----------------------------------------------------------------
php artisan serve

Expected output:
INFO  Server running on [http://127.0.0.1:8000].

=================================================================
TROUBLESHOOTING
=================================================================

If you encounter permission errors:
-----------------------------------
chmod -R 775 storage
chmod -R 775 bootstrap/cache

If images don't display:
------------------------
php artisan storage:link

If database connection fails:
-----------------------------
1. Check MySQL is running
2. Verify .env database credentials
3. Create database manually: CREATE DATABASE trip_system;

If routes don't work:
--------------------
php artisan route:clear
php artisan config:clear
